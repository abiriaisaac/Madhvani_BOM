# 🔥 Billet Reheating Furnace – Engineering BOM System

A structured **Engineering Bill of Materials (BOM) and Technical Specification System** for a steel billet reheating furnace.

Designed for:

* 🏭 Industrial Engineering (Steel Rolling Mills)
* 🔧 Maintenance & Asset Management
* 📊 Data Analysis & Optimization
* 🎓 Academic Research (Mechanical Engineering)

---

## 📱 Mobile-Friendly Design

This project is optimized for:

* ✅ Smartphones (responsive layout)
* ✅ Tablets
* ✅ Desktop browsers

You can easily view BOM data and system specifications on your phone.

---

## 🚀 Features

* 📋 Complete Furnace BOM (Single-Sheet Structured Format)
* ⚙️ Subsystem Breakdown:

  * Charging System
  * Drive System
  * Combustion System
  * Cooling System
  * Hydraulics
  * Electrical & Instrumentation
* 📊 Excel Export (OpenPyXL Automation)
* 🔢 Numeric-clean data (ready for analysis)
* 🧠 Ready for AI / ML integration

---

## 🏗️ System Overview

The system models a **Continuous Pusher Type Billet Reheating Furnace** used in steel rolling operations.

| Parameter         | Value             |
| ----------------- | ----------------- |
| Furnace Type      | Continuous Pusher |
| Temperature Range | 600 – 1250 °C     |
| Fuel              | Natural Gas       |
| Control           | PLC-Based         |
| Application       | Billet Heating    |

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/billet-furnace-bom.git
cd billet-furnace-bom
```

### 2. Install dependencies

```bash
pip install openpyxl
```

## ▶️ Usage

Run the script to generate the Excel BOM:
